# EL ORIGEN

Vertical slice de horror doméstico argentino. Entrás al departamento de tu abuela para resolver una tasación familiar y descubrís un método: cuidado usado como administración, administración usada como desgaste, desgaste usado para comprar casas por debajo de su valor.

La experiencia es point-and-click diegética: mirar, sostener, despegar, superponer, escuchar, comparar papeles y decidir qué hacer con la tasación final. La casa registra conducta del jugador y convierte esa conducta en parte del precio.

## Desarrollo

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Validación

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

`?debug=1` muestra hotspots y coordenadas sólo en desarrollo.
