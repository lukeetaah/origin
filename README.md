# EL ORIGEN

Vertical slice narrativo sobre una persona que vuelve al departamento de Nora antes de que el consorcio cambie la cerradura. La búsqueda inicial es concreta: encontrar el cuaderno azul y decidir qué hacer con el origen material que la familia simplificó.

La experiencia usa interacción física sobre objetos del departamento: iluminar, sostener, despegar, superponer, volver y comprobar consecuencias. No depende de servicios generativos ni de inicio de sesión.

## Desarrollo

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Validación

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

`?debug=1` muestra hotspots y coordenadas sólo en desarrollo.
