# EL ORIGEN

Vertical slice narrativo sobre una persona que vuelve al departamento de Elda antes de que lo vacíen. La búsqueda inicial es simple: encontrar la libreta azul y decidir qué hacer con el registro que Elda escondió.

La experiencia usa interacción física sobre objetos del departamento: iluminar, sostener, despegar, superponer, volver y comprobar consecuencias. No depende de servicios generativos ni de inicio de sesión.

## Desarrollo

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Validación

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

`?debug=1` muestra hotspots y coordenadas sólo en desarrollo.
